# Belajar Flutter Dummyjson Product

Aplikasi Flutter sederhana untuk menampilkan daftar produk dari API menggunakan package Provider dan HTTP.

Project ini dibuat untuk belajar:
- Fetch data API
- State Management Provider
- Parsing JSON
- Menampilkan data ListView
- Navigasi halaman
- Error handling
- Loading indicator

API yang digunakan:
https://dummyjson.com/products

---

# Tampilan Aplikasi

Aplikasi memiliki fitur:
- Menampilkan daftar produk
- Menampilkan detail produk
- Menampilkan gambar produk
- Loading saat mengambil data
- Error screen jika gagal mengambil data
- Refresh data

---

# Teknologi yang Digunakan

- Flutter
- Dart
- Provider
- HTTP Package
- REST API
- DummyJSON API
